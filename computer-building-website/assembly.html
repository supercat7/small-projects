PC Assembly Guide

Step-by-Step Assembly

  1. Prepare Workspace

    Start with a clean, static-free environment. Ensure you have all your components and tools organized. A large, flat surface like a desk is ideal.

  2. Install CPU

    Carefully align the CPU with the socket on the motherboard, matching the pins or notches. Gently place the CPU into the socket and secure it in place with the lever.

  3. Install CPU Cooler

    Attach the CPU cooler according to the manufacturer's instructions. Apply thermal paste (if needed), then secure the cooler to the CPU socket with screws or clips.

  4. Install RAM

    Insert the RAM sticks into the RAM slots on the motherboard. Align them properly and press down firmly until they click into place.

  5. Install Motherboard into Case

    Place the motherboard into the case, aligning the mounting holes with the standoffs inside the case. Secure the motherboard with screws and make sure it’s properly positioned.

  6. Install Power Supply (PSU)

    Install the power supply into the designated area in the case. Secure it with screws and connect the necessary cables to the motherboard and other components.

  7. Install Storage (HDD/SSD)

    Mount your storage drives (HDD or SSD) into the appropriate drive bays. Secure them with screws or tool-less clips and connect them to the motherboard using SATA cables.

  8. Install GPU

    Insert the graphics card into the PCIe slot on the motherboard. Press down firmly until it clicks into place. Secure the GPU with screws and connect any necessary power cables.
